1. 📌 Propósito da Jornada

Estou aprendendo PHP com o objetivo de me tornar um desenvolvedor FullStack que tenha um vasto conhecimento para resoluções diárias e criação de projetos do zero.

Dentro deste documento haverá várias funções, filtros e métodos que aprendi com PHP. Basicamente, é um material onde vou registrar tudo o que necessito para fixar o aprendizado e buscar como referência para novas ideias.

Pretendo desenvolver uma arquitetura no conceito de MVC para dividir tarefas e desenvolver minhas habilidades em um dos paradigmas mais importantes e utilizados da programação atualmente: a Programação Orientada a Objetos (POO).

A aplicação seguirá o padrão MVC (Model-View-Controller).

📂 Estrutura Inicial
/app
    /controllers
    /models
    /views
/core
/config
/public
/routes
🔹 Model

Responsável pela regra de negócio e comunicação com o banco de dados.

🔹 View

Responsável pela interface visual e apresentação dos dados.

🔹 Controller

Responsável por receber requisições, acionar Models e retornar Views.
